# Инструкция по загрузке проекта Modbus ESP8266 в GitHub

## Подготовка к загрузке

Прежде чем начать, убедитесь, что у вас есть:
1. Аккаунт на GitHub (https://github.com)
2. Установленный Git (https://git-scm.com/downloads) или используйте встроенный в VS Code

## Вариант 1: Загрузка через веб-интерфейс GitHub

### Шаг 1: Создание репозитория

1. Перейдите на https://github.com и войдите в свой аккаунт
2. Нажмите кнопку "New" (Новый) в правом верхнем углу
3. Введите имя репозитория: `modbus-esp8266-esp8266`
4. Добавьте описание: "Modbus TCP сервер для ESP8266 с подключением к SCADA"
5. Выберите "Public" (Публичный)
6. Поставьте галочку "Add a README file"
7. Нажмите "Create repository"

### Шаг 2: Подготовка файлов проекта

1. Скачайте и распакуйте архив проекта
2. Убедитесь, что структура папок соответствует следующей:
   ```
   modbus-esp8266-project/
   ├── README.md
   ├── requirements.txt
   ├── LICENSE
   ├── src/
   │   └── modbus.ino
   └── docs/
       └── setup_guide.md
   ```

### Шаг 3: Загрузка файлов через веб-интерфейс

1. На странице вашего репозитория нажмите кнопку "Add file" → "Upload files"
2. Перетащите файлы из папки проекта в область загрузки
3. Убедитесь, что структура папок сохранена
4. Добавьте комментарий к коммиту: "Initial commit - Modbus ESP8266 project"
5. Нажмите "Commit changes"

## Вариант 2: Загрузка через Git командную строку

### Шаг 1: Установка и настройка Git

1. Скачайте и установите Git с https://git-scm.com/downloads
2. Откройте терминал (командную строку) и выполните:
   ```bash
   git config --global user.name "Ваше имя"
   git config --global user.email "ваш.email@example.com"
   ```

### Шаг 2: Клонирование репозитория

1. На странице вашего репозитория нажмите кнопку "Code" и скопируйте URL
2. Откройте терминал и выполните:
   ```bash
   git clone https://github.com/ваш_логин/название_репозитория.git
   cd название_репозитория
   ```

### Шаг 3: Добавление файлов проекта

1. Скопируйте файлы проекта в папку репозитория
2. Откройте терминал в папке репозитория и выполните:
   ```bash
   git add .
   git commit -m "Initial commit - Modbus ESP8266 project"
   git push origin main
   ```

## Вариант 3: Использование VS Code

### Шаг 1: Открытие проекта в VS Code

1. Откройте VS Code
2. Откройте папку проекта через File → Open Folder

### Шаг 2: Инициализация Git репозитория

1. Откройте встроенный терминал в VS Code (Ctrl+`)
2. Выполните команды:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Modbus ESP8266 project"
   ```

### Шаг 3: Подключение к GitHub

1. Создайте репозиторий на GitHub (без README)
2. Следуйте инструкциям на странице репозитория:
   ```bash
   git remote add origin https://github.com/ваш_логин/название_репозитория.git
   git branch -M main
   git push -u origin main
   ```

## Структура проекта

После загрузки ваш репозиторий должен содержать следующие файлы:

```
/
├── README.md
├── requirements.txt
├── LICENSE
├── .gitignore
├── src/
│   └── modbus.ino
└── docs/
    └── setup_guide.md
```

## Описание файлов

- `README.md` - Основное описание проекта
- `requirements.txt` - Список необходимых библиотек
- `LICENSE` - Лицензия MIT
- `src/modbus.ino` - Исходный код для ESP8266
- `docs/setup_guide.md` - Подробное руководство по настройке

## Возможные проблемы и решения

### Проблема: Ошибка доступа при push
**Решение:** Используйте токен доступа вместо пароля:
1. Перейдите в Settings → Developer settings → Personal access tokens
2. Создайте новый токен с правами на repo
3. Используйте токен вместо пароля при push

### Проблема: Файлы не добавляются в коммит
**Решение:** Проверьте файл .gitignore и при необходимости добавьте:
```bash
git add --all
```

### Проблема: Конфликт при push
**Решение:** Сначала выполните pull, затем push:
```bash
git pull origin main
git push origin main
```
