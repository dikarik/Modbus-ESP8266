Modbus ESP8266 Проект
Этот проект реализует Modbus TCP сервер на базе ESP8266 для взаимодействия с SCADA системами.

Описание
Проект позволяет подключать ESP8266 к SCADA системе через протокол Modbus TCP. Реализованы следующие функции:

Чтение аналогового входа A0
Мониторинг уровня сигнала Wi-Fi
Управление светодиодом через Modbus регистры
Подключение регистров
Регистр	Назначение	Тип доступа
100	Аналоговый вход A0	Чтение
101	Уровень сигнала Wi-Fi (RSSI)	Чтение
104	Управление светодиодом (1 - Вкл, 0 - Выкл)	Чтение/Запись
Настройка
Установите необходимые библиотеки:

ESP8266WiFi
ModbusIP_ESP8266
Измените параметры Wi-Fi в файле modbus.ino:

const char* ssid = "ваш_ssid";
const char* password = "ваш_пароль";
Загрузите код в ESP8266

Использование
После загрузки и подключения к Wi-Fi, ESP8266 будет доступен по протоколу Modbus TCP на порту 502. Используйте SCADA систему для чтения регистров 100, 101 и управления регистром 104.
