# Как создать репозиторий на sourcecraft.dev

## Подготовленные файлы

Все необходимые файлы для репозитория уже подготовлены в папке `modbus-esp8266-project`:

1. `README.md` - Основное описание проекта
2. `src/modbus.ino` - Исходный код для ESP8266
3. `docs/setup_guide.md` - Подробное руководство по настройке
4. `requirements.txt` - Список необходимых библиотек
5. `LICENSE` - Лицензия MIT
6. `.gitignore` - Файл игнорирования

## Шаги для создания репозитория

1. Перейдите на сайт https://sourcecraft.dev/kitdimon/

2. Создайте новый репозиторий с названием, например: `modbus-esp8266-esp8266`

3. Загрузите подготовленные файлы из папки `modbus-esp8266-project`:
   - Загрузите содержимое папки `modbus-esp8266-project` в корень репозитория
   - Убедитесь, что структура папок сохранена:
     ```
     /
     ├── README.md
     ├── requirements.txt
     ├── LICENSE
     ├── .gitignore
     ├── src/
     │   └── modbus.ino
     └── docs/
         └── setup_guide.md
     ```

4. Добавьте описание репозитория:
   ```
   Проект Modbus TCP сервера на ESP8266 для взаимодействия с SCADA системами.
   Реализует чтение аналогового входа, мониторинг Wi-Fi и управление светодиодом через Modbus регистры.
   ```

5. Выберите лицензию MIT (она уже включена в файл LICENSE)

6. Сделайте репозиторий публичным, чтобы другие разработчики могли использовать ваш код

## После создания репозитория

После создания репозитория вы сможете:
- Делиться кодом с другими разработчиками
- Получать обратную связь и предложения по улучшению
- Отслеживать изменения в коде
- Создавать релизы проекта

Если у вас возникнут вопросы по настройке проекта, обратитесь к документации в папке `docs`.
